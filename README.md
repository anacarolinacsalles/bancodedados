Descrição do problema: 
A gestão de eventos envolve diversas atividades como cadastro de participantes, organização de 
shows, venda de ingressos e controle de pagamentos. Em eventos que possuem apresentações 
artísticas, é necessário administrar informações sobre clientes, artistas, ingressos, setores do 
evento e compras realizadas. Dessa forma, um banco de dados é essencial para organizar essas 
informações, garantir o controle de capacidade do evento, registrar transações financeiras e facilitar 
a administração geral do sistema.  

Justificativa da escolha: 
O tema gestão de eventos foi escolhido por representar um cenário real e comum, utilizado em 
festivais, shows e apresentações culturais. Esse tipo de sistema exige o gerenciamento de várias 
entidades e relacionamentos, como clientes, artistas, eventos e ingressos, tornando-o adequado 
para aplicação de conceitos de modelagem de banco de dados, como hierarquias, relacionamentos 
N:M e diferentes tipos de atributos.
